# Convert Sources Image -> Sheet (Plan)

You said you captured a screenshot/photo of sources and want it converted into an Excel or Google Sheet.

## What I Need From You

To do the conversion, I need the image itself:

- upload the screenshot/photo here, or
- paste the file path to the image on disk

## Output Format (Recommended Columns)

Once I have the image, I will extract it into a table with columns like:

- `source_name`
- `type` (dataset / API / report / tool / article)
- `domain` (shipping / ports / weather / legal / finance)
- `link`
- `notes`
- `access` (free / paid / unknown)
- `relevance` (MVP / later)

## Where It Will Go

- Excel file: stored in the Denkern workspace
- optionally also exported as CSV for Google Sheets import
