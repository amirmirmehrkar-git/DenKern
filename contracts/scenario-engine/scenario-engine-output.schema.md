# Scenario Engine Output Schema

## Purpose

Defines operational response scenarios generated by the backend decision-support layer.

## Producer

Scenario Engine

## Consumers

- Recommendation Layer
- Dashboard

## Schema

```json
{
  "shipment_id": "SHIP-001",
  "scenarios": [
    {
      "scenario_id": "WAIT",
      "name": "Wait for shipment",
      "expected_delay_days": 5,
      "action_cost_eur": 0,
      "production_loss_eur": 750000,
      "total_cost_eur": 750000,
      "risk_level": "HIGH"
    },
    {
      "scenario_id": "EXPEDITE",
      "name": "Expedite from Amsterdam",
      "expected_delay_days": 2,
      "action_cost_eur": 200000,
      "production_loss_eur": 300000,
      "total_cost_eur": 500000,
      "risk_level": "MEDIUM"
    }
  ]
}
```

## Notes

- Backend generates scenarios.
- ML model does NOT make operational decisions.
