# Team Message: Shift From Prediction To Decisions

Right now we are still showing predictions.

We need to move to explicit decision scenarios with cost comparison and a clear recommendation.

The shift is:

```text
Prediction -> Visualization
```

to:

```text
Decision -> Recommendation -> Action
```
