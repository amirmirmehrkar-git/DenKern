# Miro AI / Sidekick Prompt - Final Cleanup Only

Copy/paste this directly into **Miro AI / Sidekick**:

```text
Update the current DenkKern Lena 2.0 Miro board.

Goal:
Make final cleanup only. Do not redesign the board. Do not add new product concepts. Do not add more complexity.

Apply only these three improvements:

1. Reduce visual density

Increase whitespace between major phases.
Make phase labels larger and easier to read.
Move secondary sticky notes slightly away from the main flow.
Keep the main flow visually clean.
Do not delete important content.
Do not add new architecture layers.

2. Strengthen the final outcome

At the end of the main flow, add a strong green outcome block:

“Production downtime avoided”
“€450k saved”
“Operational confidence increased”

Make this the final visible business outcome of the journey.

3. Separate executive view from internal system view

Create two clearly labeled areas:

BOARD A — Executive Flow
For customers, mentors, and investors.
Include only:
- Lena journey
- delayed shipment problem
- financial impact
- scenario comparison
- recommendation
- human decision
- production saved

BOARD B — Internal System Map
For the founding team and technical discussion.
Include:
- APIs
- ML prediction
- mock ERP
- mock freight API
- scenario engine
- financial impact engine
- recommendation logic
- audit trail
- dependencies

Keep Board A visually simpler and larger.
Keep Board B more detailed but secondary.

Add one final red note near the board:

“After this cleanup: freeze the board and validate with customers.”

Final check:
A new viewer should understand the product story in under 30 seconds.
The board should be clean, readable, and ready for MVP week.
```
